Detailed Description

A placeholder for Ruby exceptions.

You can use this to safely throw a Ruby exception using C++ syntax:

VALUE foo(VALUE self) {
RUBY_TRY {
throw Rice::Exception(rb_eMyException, "uh oh!");
RUBY_CATCH
}
